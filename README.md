> *"Security is not a product to be installed — it's an architecture to be designed."*

I'm a cybersecurity architect and technology leader protecting the infrastructure that connects **tens of millions of subscribers** across Bangladesh. At **Robi Axiata PLC** — Bangladesh's second-largest mobile operator and an Axiata Group subsidiary — I architect security strategy, govern risk, and lead the security assessment program across complex IT–Telco environments.

My career bridges two worlds: **a decade of software engineering discipline** refined into **enterprise security leadership**. That dual fluency means I don't just assess risks — I understand how systems break, how code gets exploited, and how to build security into architecture from the ground up.

Outside corporate security, I bring this perspective into education as **Adjunct Faculty at United International University**, where I teach a DevSecOps-integrated Software Engineering Lab curriculum designed to produce security-conscious engineers from day one.
